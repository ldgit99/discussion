# 토론모임 사용 가이드

중학생 4-5명이 한 모둠으로 AI 보조와 함께 토의하는 도구입니다.

## 교사를 위한 안내

### 첫 사용

1. **회원가입** — 이메일 + 비밀번호. 역할은 "교사"로 선택.
2. **교사 권한 부여** (필요 시): 관리자에게 본인 이메일을 알려주세요. (운영 단계에서는 학교 도메인 자동 인증)
3. 로그인 후 **교사 대시보드** 진입.

### 한 차시 운영

1. **[새 수업 만들기]** 클릭
   - 주제 입력 (5자 이상, 학생들이 보게 될 핵심 질문)
   - 학생 수 + 모둠당 인원 (보통 4-5명)
   - 제한 시간
2. **모둠 코드 발급** — 자동으로 모둠 N개가 만들어지고 6자리 코드가 표시됩니다.
3. **학생에게 코드 배포** — 인쇄·저장 버튼으로 출력하거나 칠판에 안내.
   - 같은 모둠을 원하는 친구들끼리 같은 코드 입력
4. **모니터링** — 학생 입장 인원이 실시간 갱신됩니다.
5. **결과 확인** — 학생이 [우리 모둠 의견 제출하기] 누르면 하단 결과 보드에 즉시 등장.

### AI는 무엇을 하나요

학생 대신 답을 만들지 않습니다. **학생의 사고와 판단을 돕는 6가지 역할만** 합니다:

- **정리하기** — 학생 발화를 그대로 요약
- **의견 비교** — 공통점·차이점·기준 (어느 의견이 낫다고 말하지 않음)
- **근거 확인** — "왜 그렇게 생각해?" 같은 질문 던지기
- **합의 돕기** — 공통점·남은 쟁점 정리
- **질문 만들기** — 토의가 끊기면 새 질문 제안
- **태도 점검** — 무례한 표현 감지 시 부드러운 환기

학생의 의견을 평가하거나 합의문을 직접 작성하지 않습니다.

### 개인 채팅 사후 조회

학생은 토의 도중 [팀] / [개인] 탭을 오갈 수 있습니다. 개인 탭에서는 AI 코치와 1:1로 생각을 정리합니다.

- 기본값: **교사도 볼 수 없음**.
- 학생이 [교사 열람 허용]을 클릭하면, 토의 종료 후 교사가 모둠 상세 페이지의 "열람" 버튼으로 사후 확인 가능.

## 학생을 위한 안내

### 가입 + 입장

1. **회원가입** — 이메일 + 비밀번호 + 반/번호/이름.
   - 모둠에서는 `반-번호-이름`(예: `3-15-김지혜`)으로 표시됩니다.
2. 로그인 후 **선생님이 알려준 6자리 코드** 입력.
3. **[입장하기]** 클릭 → 모둠 입장.

### 토의방 사용

화면은 3개 영역으로 나뉩니다.

**왼쪽 — 채팅 (팀 / 개인 탭)**
- **팀 탭**: 모둠 전원 + AI 진행자가 함께 보는 방.
- **개인 탭** 🔒: 나와 AI 코치만 보는 사적인 공간. 생각을 정리할 때 사용.
  - 개인 메시지 옆 [팀에 공유] 버튼 → 팀 채팅에 보냄.

**가운데 — 공동 토의 보드**
- [+] 버튼으로 의견 카드 등록 (의견 + 근거).
- 의견 비교 / 선택 기준 / 쟁점 / 대표 의견 후보 섹션을 함께 작성.

**오른쪽 — AI 보조 패널**
- 6개 버튼: 정리하기, 의견 비교, 근거 확인, 질문 만들기, 태도 점검, 합의 돕기
- 개인 탭에서는 [내 생각 정리하기] 버튼이 추가됨.

### 마무리

1. 합의가 되면 하단의 **모둠 토의 결과**를 채워 넣기.
2. **[우리 모둠 의견 제출하기]** 클릭 → 선생님 대시보드에 즉시 표시.

## 자주 묻는 질문

**Q. AI가 답을 다 정해주나요?**
A. 아니요. AI는 학생의 발화를 기반으로만 요약·비교·질문합니다. 어느 의견이 낫다고 말하거나 합의문을 직접 작성하지 않습니다.

**Q. 개인 채팅이 정말 안 보이나요?**
A. 본인만 볼 수 있습니다. 교사도 본인이 [교사 열람 허용]을 켜기 전까지는 볼 수 없습니다.

**Q. 무례한 말을 쓰면 어떻게 되나요?**
A. AI가 자동으로 감지해서 "이 표현은 다른 친구의 마음을 다치게 할 수 있어요" 같은 안내를 모둠 채팅에 보냅니다. 작성자를 호명하거나 비난하지 않습니다.

**Q. 모둠 정원이 다 찼어요.**
A. 다른 모둠 코드를 사용하세요. 선생님이 코드 여러 개를 안내했을 거예요.

**Q. 닉네임을 바꾸고 싶어요.**
A. 닉네임은 가입 시 정해진 `반-번호-이름`으로 자동 설정됩니다. 변경하려면 회원가입을 다시 해야 합니다 (운영 단계에서 개선 예정).
